# 组件样式提示词

复制下面提示词给任意 AI 使用。

```text
请按 监管平台规范优化当前页面组件样式。

请优先参考：
- ai-guides/dorami-ui-protocol-guide.md
- ai-guides/supervision-page-builder-guide.md
- data/component-catalog.json
- data/component-library.json
- data/design-token.json
- knowledge/02-Component-DNA.md
- knowledge/04-Visual-Rules.md

要求：
1. 基础控件遵守 Dorami 组件协议。
2. 静态 Demo 使用 dcp-* 类名表达组件协议。
3. 页面样式使用 --theme-* token。
4. 组件样式使用 --dcp-* token。
5. 图标使用 assets/icons/。
6. 英文和数字可使用 assets/fonts/D-DIN-PRO-*.otf。
7. 不要使用临时颜色、临时圆角、临时阴影。

请输出：
- 修改建议
- 组件映射
- 样式代码
- 自检结果
```

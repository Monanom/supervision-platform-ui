# Golden Demo Checklist

Run this checklist before final delivery.

## Golden Demo Alignment

- [ ] Did you read `golden-demo/supervision-warning-supervise-query.html` before generating?
- [ ] Did you reuse the same shell model: fixed sidebar, translucent top bar, adaptive right content?
- [ ] Did you keep the right content background image plus Brand-1 base color?
- [ ] Did you use the bundled platform Logo?
- [ ] Did you use D-DIN-PRO for English letters and numbers?

## Component Alignment

- [ ] Do buttons follow Dorami protocol sizing and states?
- [ ] Are filter Select/Input/Button controls 32px high?
- [ ] Do Select/Dropdown controls use custom popup styling with 4px option spacing?
- [ ] Does the table follow parent width and keep only the action column fixed?
- [ ] Are table body cells top-aligned with 14px vertical padding?
- [ ] Are table lifecycle statuses dot plus text rather than Tag visuals?
- [ ] Does each row have at most one primary action button?
- [ ] Are danger actions outline/light with danger text, not solid red?
- [ ] Do modals use icon-only close controls and 32px footer buttons?

## Visual Alignment

- [ ] Are metric cards frosted, while list/work cards stay plain white?
- [ ] Are cards 8px radius or less, and small data cards 4px where applicable?
- [ ] Is the sidebar selected state white 12% overlay plus indicator?
- [ ] Are hover/focus/disabled states present for visible controls?
- [ ] Are all functional icons from bundled assets?

## Anti-Pattern Check

- [ ] No marketing hero or landing-page layout.
- [ ] No command-center/dark big-screen treatment.
- [ ] No arbitrary gradients, neon shadows, or decorative blobs.
- [ ] No generic SaaS card-heavy dashboard composition.
- [ ] No un-tokenized colors, shadows, radii, fonts, or icons.

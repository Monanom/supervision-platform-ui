# Static Demo Template

Use this folder when generating static HTML/CSS/JS demos.

Recommended process:

1. Start from `supervision-static-demo-template.html`.
2. Replace page title, metrics, filter fields, table columns, mock data, and row actions.
3. Keep the shell, token block, asset references, component class hooks, and interaction patterns.
4. Compare the result with `../golden-demo/supervision-warning-supervise-query.html`.
5. Run `../compiler/golden-demo-checklist.md`.

Do not start from a blank page unless the requested page type is not covered by the template.

# 00 Core Principles

P0 Business First：业务表达优先于视觉表达，禁止改变业务流程、删除字段、隐藏重要信息。

P1 Decision First：页面必须帮助监管人员判断当前情况、异常位置和处理方式。

P2 Information First：信息密度优先于装饰，不得为了留白减少信息。

P3 Consistency First：一致性高于创意，优先复用已有 Pattern 和组件。

P4 Pattern Before Creativity：AI 必须先选择页面 Pattern，再生成页面。

P5 Visual Restraint：监管平台保持克制、专业、可信，禁止营销风、电商风、毛玻璃、大渐变、胶囊按钮。

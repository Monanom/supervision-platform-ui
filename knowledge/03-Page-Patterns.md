# 03 Page Patterns

Dashboard:
PageHeader → StatisticPanel → TrendChart / AnalysisPanel → BusinessTable

List:
PageHeader → SearchPanel → StatisticPanel(optional) → Toolbar → DataTable → Pagination

Detail:
PageHeader → OverviewCard → DescriptionList → RelatedTable / Timeline

Form:
PageHeader → FormCard → FormGroup → ActionBar

Dialog:
DialogTitle → Content → FooterAction

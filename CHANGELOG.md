# CHANGELOG

## v1.0.0 - 2026-07-10

首次发布 Supervision KB。

包含内容：
- 监管平台页面规范知识库。
- 任意 AI 可读的 `ai-guides/`。
- 可复制使用的 `ai-prompts/`。
- Codex 可选安装的 `codex-skills/`。
- 监管平台常用视觉资源 `assets/`。

使用者更新方式：

- 使用 Codex skill 的用户重新复制 `codex-skills/` 下的两个 skill 到 `~/.codex/skills/`。
- 重启 Codex 后生效。
